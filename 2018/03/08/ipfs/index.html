IPFS

ipfs节点创建

以下均在mac系统下操作

  • 下载ipfs安装压缩包 下载链接：https://ipfs.io/

  • 安装 打开终端，cd进入你下载压缩包的目录下 运行如下命令进行解压缩

$ ls
0305ipfs				0305资料.zip
0305ipfs.zip				go-ipfs
0305案例				go-ipfs_v0.4.13_darwin-amd64.tar.gz
0305资料				ipfs-http-demo
0305案例 2				ipfs-http-demo-数据上传下载.zip
0305案例.zip				ipfs-http-demo.zip
$ tar xvfz go-ipfs_v0.4.13_darwin-amd64.tar.gz
x go-ipfs/build-log
x go-ipfs/install.sh
x go-ipfs/ipfs
x go-ipfs/LICENSE
x go-ipfs/README.md

当然你也可以直接打开finder窗口找到go-ipfs_v0.4.13_darwin-amd64.tar.gz这个压缩文件双击进行解压 解压后得到go-ipfs文件包，cd进入该文件包内

$ ls

接着执行下面的命令，将ipfs文件移到/usr/local/bin/ipfs目录下

mv ipfs /usr/local/bin/ipfs

当然你也可以在终端通过如下命令打开/usr/local/bin目录对应的finder窗口

$ open /usr/local/bin

$ ipfs add file.txt
added QmVtFbBrZQtQVcVRZuUtUiTsBkNgqBzM72uCoWsJT9RHUT file.txt
$ ipfs cat QmVtFbBrZQtQVcVRZuUtUiTsBkNgqBzM72uCoWsJT9RHUT
my name is yang

执行完这个命令后，会将file.txt文件添加到ipfs当前的本地节点上，并对file.txt文件生成一个唯一的hash值QmVtFbBrZQtQVcVRZuUtUiTsBkNgqBzM72uCoWsJT9RHUT 可以通过ipfs cat QmVtFbBrZQtQVcVRZuUtUiTsBkNgqBzM72uCoWsJT9RHUT查看本地ipfs节点上的file.txt文件 注意不是用ipfs cat file.txt
